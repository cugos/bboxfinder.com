bboxfinder.com
==============

Helper page for finding bbox values from a map to help with interaction with tools like gdal, leaflet, openlayers, etc.

## For contributors

To fix bugs, make, or propose changes, fork this repo, file a pull request.

Browse or file issues from the [Issues tab](https://github.com/cugos/bboxfinder.com/issues).

Note: changes to the app don't deploy automatically. After the change is reviewed and merged into the `main` branch, an admin must deploy the build.

## For admins

See [DEPLOYMENT.md](DEPLOYMENT.md) for information on how deployment works.
